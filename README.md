# Jarvis Windows AI Assistant

Голосовой ИИ-ассистент Jarvis для Windows.

## Возможности
- Голосовое управление
- Интеграция с современными LLM
- Управление системой Windows
- Навыки (skills)

## Установка

```bash
git clone https://github.com/bio8804/Jarvis-Windows-AI.git
cd Jarvis-Windows-AI
pip install -r requirements.txt
python main.py
```
